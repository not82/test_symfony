Lancer docker en arriète plan
> docker-compose up -d

Vérifier les services Docker 
> docker-compose ps

Logs Docker
> docker-compose logs

Lancer le serveur local
> symfony server:start -d

Avoir les erreurs Apache / PHP en temps réel
> symfony server:log

Lancer des commandes psql 

> symfony run psql
